# Czarina Agoncillo Personal Brand Site

A high-end Next.js 14 editorial website for Czarina Agoncillo (Developer, SEO Specialist, Systems Builder, Author).

## Setup
1. `npm install`
2. `npm run dev`

## Build & Export
1. `npm run build`
2. Static output in `/out` folder.

## Deployment
Automatic deployment via GitHub Actions in `.github/workflows/deploy.yml`.
